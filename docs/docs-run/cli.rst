============
OctaDist CLI
============

Running command-line
--------------------

.. code-block:: sh

    # Open help page
    octadist_cli -h

    # Compute parameters
    octadist_cli -i INPUT

    # Compute parameters and show formatted output
    octadist_cli -i INPUT -o

    # Compute parameters and save output as file
    octadist_cli -i INPUT -s OUTPUT
